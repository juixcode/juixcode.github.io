Pre-cook Items

1st Command

/execute at @e[type=item,nbt={Item:{id:"minecraft:<raw_item>",Count:1b}}] run summon item ~ ~ ~ {Item:{id:"minecraft:<cooked_item>",Count:1b}}

Explication

It summons the cooked item at the raw item.

2nd Command

/execute at [type=item,nbt={Item:{id:"minecraft:<raw_item>",Count:1b}}] run summon item ~ ~ ~ {Item:{id:"minecraft:<raw_item>",Count:1b}}] run summon minecraft:experience_orb ~ ~ ~ {Value:1}

Explication

It gives experience like a furnace.

3rd Command

/kill [type=item,nbt={Item:{id:"minecraft:<raw_item>",Count:1b}}]

Explication

It removes the useless raw item after beeing cooked.

Tested in versions 1.16, 1.18
